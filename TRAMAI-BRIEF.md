# TRAMAI — Personal AI Production Desk
## Claude Code Build Brief
> **Project name:** TRAMAI (from Italian "trama" = weave/plot/fabric + AI)
> **Owner:** Roberto — Senior Art Director, Berlin
> **Machine:** MacBook Pro 2013 (16GB RAM, macOS Big Sur) — app runs in browser only, no local processing
> **Budget:** €30/month max for APIs
> **Goal:** Personal node-based tool connecting free/cheap AI backends through one clean, dark interface
> **Deployment:** Vercel (free tier) — accessible via `tramai.vercel.app`
---
## 1. PROJECT PHILOSOPHY
This is NOT a SaaS product. It's a personal creative workbench — like a carpenter's custom jig.
The key differentiator from tools like Weavy/Flora: **nodes speak Art Director language, not ML language.** Instead of a "Stable Diffusion node" with CFG scale and sampler settings, there's a "Lighting Direction node" that translates concepts like "golden hour, soft fill" into the right technical parameters for whichever backend is selected.
Small node library (12 nodes), opinionated defaults, clean dark UI, zero complexity tax.

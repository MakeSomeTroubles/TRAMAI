## SHORT DESCRIPTION (2-3 righe, per portfolio grid)

**English:**
TRAMAI is a visual workflow tool that connects AI image generation with art direction thinking. Built for advertising professionals who need to produce, iterate, and refine AI-generated visuals at campaign scale — without losing creative control.

**Deutsch:**
TRAMAI ist ein visuelles Workflow-Tool, das KI-Bildgenerierung mit Art-Direction-Denken verbindet. Entwickelt für Werbeprofis, die KI-generierte Visuals in Kampagnenqualität produzieren, iterieren und verfeinern müssen — ohne die kreative Kontrolle zu verlieren.

## MEDIUM DESCRIPTION (per case study page)

**English:**
As a Senior Art Director working with generative AI daily, I kept running into the same problem: existing AI tools are built for individual images, not for advertising production. There's no way to think in systems — to connect a brief to a prompt to a generation to a variation to a final asset, all in one visual space.

TRAMAI solves this. It's a node-based workspace where each step of the AI production pipeline is a visual block you can connect, configure, and control. Write a brief, connect it to a prompt template optimized for your ad format, generate images through Fal.ai, create variations, upscale, color grade, and preview everything on a wall — all in one flow.

The name comes from the Italian "trama" — the weft thread that weaves through a fabric. TRAMAI weaves AI generation into a structured creative process.

Built with React, React Flow, Tailwind CSS, and Zustand. Connects to Fal.ai for image generation.

**Deutsch:**
Als Senior Art Director, der täglich mit generativer KI arbeitet, bin ich immer wieder auf dasselbe Problem gestoßen: Bestehende KI-Tools sind für Einzelbilder gebaut, nicht für Werbeproduktion. Es gibt keine Möglichkeit, in Systemen zu denken — ein Briefing mit einem Prompt mit einer Generierung mit einer Variation mit einem finalen Asset zu verbinden, alles in einem visuellen Raum.

TRAMAI löst dieses Problem. Es ist ein node-basierter Workspace, in dem jeder Schritt der KI-Produktionspipeline ein visueller Block ist, den man verbinden, konfigurieren und kontrollieren kann. Briefing schreiben, mit einem für das Werbeformat optimierten Prompt-Template verbinden, Bilder über Fal.ai generieren, Variationen erstellen, hochskalieren, Color Grading anwenden und alles auf einer Preview Wall betrachten — alles in einem Flow.

Der Name kommt vom italienischen „trama" — der Schussfaden, der durch ein Gewebe läuft. TRAMAI verwebt KI-Generierung in einen strukturierten kreativen Prozess.

Entwickelt mit React, React Flow, Tailwind CSS und Zustand. Verbunden mit Fal.ai für Bildgenerierung.

## FEATURE LIST (per case study page)

- **Node-based visual workflow** — Connect AI production steps like building blocks
- **Advertising-native prompt templates** — Pre-built for product shots, lifestyle, UGC, headlines
- **Multi-model pipeline** — Brief → Prompt → Generate → Variations → Upscale → Export
- **Preview Wall** — See all generated assets side by side
- **Batch generation** — Produce multiple variations in one flow
- **Art direction controls** — Lighting, composition, color mood as configurable parameters
- **Built for iteration** — Change one variable, regenerate, compare
